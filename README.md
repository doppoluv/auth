# Сервис аутентификации (auth)

Это сервис для аутентификации на базе gRPC, реализующий регистрацию, вход и проверку прав администратора.

**Ключевые возможности**
- Регистрация пользователей (`Register`)
- Вход и получение JWT (`Login`)
- Проверка, является ли пользователь админом (`IsAdmin`)

## Конфигурация

Сервис читает конфигурацию (см. `config/example.yaml`) из YAML-файла. Основные параметры:
- `env` - окружение (`local`, `production`)
- `storage_path` - путь к локальной БД/хранилищу
- `token_ttl` - время жизни токена
- `grpc.port`, `grpc.timeout` - параметры gRPC
